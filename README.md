***Setting up***
---
**Pre-requisites**
- git
- python3 with pip3.
---
**Installing**  
  
```git clone https://github.com/HeapHeapHooray/ShadowCandii```  
```cd ShadowCandii```  
```pip3 install konsave```  
```konsave -i ShadowCandii.knsv```  
```konsave -a ShadowCandii``` 

*Relog or reboot.*
